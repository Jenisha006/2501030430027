🍳 RecipeAI – AI-based Recipe Generator

📌 Project Overview

RecipeAI is an AI-based Recipe Generator website where users can input ingredients they have at home and get personalized recipe suggestions based on their dietary preferences and available ingredients.

---

📁 Repository Structure

📁 Figma/  → 9 screen designs exported as PNG screenshots
📁 Code/   → Website source code (HTML, CSS, JS)

---

🖥️ Screens Included

**Screens**:
1. Home / Landing Page
2. Sing Up
3. Login
4. Ingredient Input
5. Recipe Results
6. Recipe Detail
7. Saved Recipes
8. About & Contact
9. AI Generating (Loading Screen)

---

🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript

---

🚀 How to Run

1. Download the Code folder
2. Open index.html in any browser
3. No installation needed!
